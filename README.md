# Patchbot!
> A rewrite of my personal Discord bot in rust. :)

## Prerequisites

- All API keys listed in .env.example
- [Rust](https://rustup.rs/)

## Setup

1. Clone this repo
   ```bash
   git clone https://github.com/twigform/patchbot-rs
   cd patchbot-rs
   ```

2. Copy .env.example file and fill in all your keys
   ```bash
   cp .env.example .env
   ```

3. Run it!
   ```bash
   cargo run
   ```

## Commands

| Command | Description |
|---------|-------------|
| `>help` | Gives a list of commands. (Can also be used for info on a specific command.) |
| `>ping` | pong :) |
| `>rgif` | Sends a (kind of) random gif. |
| `>cat` | Random cat. |
| `>consequence` | Scary... |

Feel free to PR a command or feature if you'd like!
